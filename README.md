# GEX Dash

Gamma Exposure dashboard for SPX and ES using the Schwab Market Data API.

## Setup

1. Copy `.env.example` to `.env` and fill in your Schwab API credentials
2. Create a virtual environment: `python -m venv .venv`
3. Activate it: `.venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Run: `python src/auth.py`
